# PoliceRegionalSafetyAdvisory
This is an Indian police website for giving civilians information about safety guidelines

This website is still in development phase ,partial frontend have been done using simple html,css and javascript and css framework tailwind also have been 
used for styling. I'm planning on implementing React JS for more rich User experiance later on.
vite is also used to create a self reloading server for my project and i have added a script "start" to immediately run vite.
No backend has been done yet .

//HOW TO START?
after opening the project and installing necessary node modules,simply run command "npm start" to launch the website.
